# 🚗 Vehicle Surveillance & Parking Management System

##  Overview

The **Vehicle Surveillance & Parking Management System** is a smart campus-based application built using **Java and Object-Oriented Programming (OOP)** principles.

It simulates a real-world parking and surveillance system where students, faculty, and security personnel interact with a centralized platform to manage parking efficiently and ensure compliance with safety rules.

This project highlights strong **problem-solving skills**, **system design thinking**, and **practical implementation of OOP concepts**.

---

##  Key Features

###  Smart Parking Management

* Allocate and manage parking slots
* Track vehicle entry and exit
* Calculate parking fees based on duration

###  Reservation System

* Reserve parking slots in advance
* Automatic expiry of reservations

###  Carpooling System

* Setup shared rides
* Get discounts for eco-friendly usage

###  Loyalty Rewards System

* Earn points based on usage
* Redeem rewards like free parking

###  Violation Detection

* Overspeeding detection
* Helmet rule enforcement
* Illegal parking alerts

###  Fine Management

* Automatic fine generation
* View and pay fines

###  Role-Based Access Control

* Student
* Faculty
* Guard
